# 프롬프트 모음 폴더

이 폴더에는 홈페이지 개발 과정에서 사용된 모든 프롬프트가 저장되어 있습니다.

## 파일 구조

- **프롬프트-전체.md**: 모든 변경사항을 카테고리별로 정리한 요약본
- **프롬프트-개별-시간순.md**: 사용자가 보낸 개별 프롬프트를 시간순으로 정리한 상세본
- **자동-저장-스크립트.md**: 프롬프트 자동 저장 가이드
- **README.md**: 이 파일 (폴더 설명)

## 자동 저장 방법

프로그램 종료 전에 프롬프트를 저장하려면:

```powershell
# 프로젝트 루트에서 실행
.\자동-프롬프트-저장.ps1
```

또는 수동으로 현재 세션의 대화 내용을 복사하여 `프롬프트-세션-[날짜].md` 파일로 저장하세요.
