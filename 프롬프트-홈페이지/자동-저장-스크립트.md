# 프롬프트 자동 저장 가이드

## 사용 방법

프로그램을 종료하기 전에 다음 명령을 실행하세요:

```bash
# 프롬프트 폴더로 이동
cd "프롬프트-홈페이지"

# 또는 프로젝트 루트에서
.\프롬프트-홈페이지\자동-저장.ps1
```

## PowerShell 스크립트 (자동 저장)

프로젝트 루트에 `자동-프롬프트-저장.ps1` 파일을 생성하여 사용할 수 있습니다.

## 수동 저장 방법

1. 현재 세션의 모든 대화 내용을 복사
2. `프롬프트-홈페이지/프롬프트-세션-[날짜].md` 파일로 저장
3. 또는 기존 파일에 추가

## 주의사항

- 세션 종료 전에 반드시 프롬프트를 저장하세요
- 중요한 변경사항은 즉시 저장하는 것을 권장합니다
